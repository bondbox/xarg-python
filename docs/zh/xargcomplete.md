xargcomplete
============

`xarg` 可为所有基于 `argparse` 和 [argcomplete](https://kislyuk.github.io/argcomplete/) 编写的 `Python` 命令行程序提供参数补全管理。

要求
----

- `bash` 或者 `zsh`
- 包依赖于 `xarg`，并且通过 `pip` 来管理

启用命令补全
------------

```shell
xargcomplete init
```

更新命令补全
------------

```shell
xargcomplete update
```
